# Rnwolfe Tap

## How do I install these formulae?

`brew install rnwolfe/tap/<formula>`

Or `brew tap rnwolfe/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "rnwolfe/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
